列表的定义

[1,2,3,4,5,6]

list1 = [1,2,3,4,5,6]

索引值

在列表中,索引是从0开始的

从左往右数:0 1 2 3…

从右往左数 -1 -2 -3 -4

左开右闭 #左边可以获取到,右边不能获取到

切片

获取列表里面的值

[开始位:结束位:步长] # 步长: 指开始值后第几个在取值 步长为-1的时候表示从后往前获取

列表的基本操作

列表的增删改查,统计,排序

l = [0,1,2,3,4,5,6,7,8]

l2 = [“python”,’hello’]

列表的增加方法

.insert(索引值,数据) 在指定位置插入数据

.append(数据) 在列表的末尾添加数据

.extend([列表]) 能够添加的数据为: 可以用for循环遍历的数据

列表的修改

列表[索引] = 数据 # 修改指定位置的数据

列表的删除

l = [0,1,2,3,3,5,6,7,8]

.remove(数据) 删除第一个出现的数据

.pop(索引值) 删除指定索引值数据,如果没有数据,默认删除末尾数据

.clear() 清空列表

del l[索引] # 删除指定位置的数据

列表的统计方法

l = [0,1,2,3,3,5,6,7,8]

len(列表) 统计列表的长度

count(数据) 统计数据在列表中出现的次数

列表排序

l = [4,2,4,6,2,3,5,7,1,10]

.sort() 对列表升序排序

.sort(reverse=True) 降序排序 reverse 反转

.reverse() 反转,只是先后位置调换位置

.intex(数据) 查找的数据在列表中第一次出现的索引

enumerate() 返回列表的位置和数据
