1变量

1.1变量定义

  • 在 Python 中，每个变量 在使用前都必须赋值，变量 赋值以后 该变量 才会被创建

  • 等号（=）用来给变量赋值

    • = 左边是一个变量名

    • = 右边是存储在变量中的值

变量定义之后，后续就可以直接使用了

1.2变量的命名和使用

在python中使用变量时，需要遵守的一些规则和指南

  • 变量名只能包含字母，数字和下划线。变量名可以字母或下划线开头，但是不能以字母开头，例如 message_1可以做变量命名，但是1_message就不可以
  • 变量名不能包含空格，但可以使用下划线来分割其中的单词，例如greeting_greeting是可以的，但是greeting greeting就会应发报错
  • 不要将python的关键词和函数作为变量名，也不要使用python保留用于特殊用途的单词，如 print det等，详见下面补充
  • 变量名应该简单又具有描述性
  • 慎用小写字母l和大写字母O，因为容易看成1何0
